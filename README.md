# healthkit_extensions_flutter

Flutter package for convenient HealthKit-related extensions.

## Project Structure

- `lib/`: package source code
- `test/`: package unit tests
- `example/`: sample Flutter app target for local manual testing

## Getting Started

```bash
flutter pub get
flutter test
```

## Run Sample App

```bash
cd example
flutter pub get
flutter run
```

The sample app imports this package via a local path dependency (`../`).
